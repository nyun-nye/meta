# 상품 목록 페이지 요구사항 분석 및 기능 목록

## 📌 기본 정보

### 프로젝트명:

신발 쇼핑몰 상품 목록 페이지 (웹, 모바일)

### 사용하게 될 기술:

React

### 문서 작성자: 윤예진

## 📝 고객 요구사항 정리

- 모바일 UI
- 상품 사진
- 장바구니에 담긴 상품 갯수 실시간 업데이트
- 배포 URL

## 📋 기능 목록

- 이미지가 포함된 상품 카드
- 2개의 컬럼으로 구성된 상품 목록 레이아웃
- 상품 카드에서 담기 버튼과 담김 상태 변화
- 담기 버튼을 누른 후, 헤더에 담긴 상품의 갯수 변화
